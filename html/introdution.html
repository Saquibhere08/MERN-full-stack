HTML(hyper-text markup Language)

    1.Tags: An HTML tag is a special keyword enclosed in angle brackets (< >) that is used to define and structure elements on a web page. 
        1. < >  : Opening Tag
        2. < / > : closing Tag
        3. < /> : Self-Closing Tag  
            Note. For every opening tag there should be an closing tag also. 
            
    2. Element: opening Tag + content + Closing tag
    3. Attribute :  * Attribute is something which will be providing in order to give additional data or insights for an 
                    for that particular specific tag.
                    * It is present in the opening tag.

    4. Comment: * An HTML comment is used to add notes or explanations in the HTML code that are not displayed on the web page by the browser.
                * For adding an comment we can use ctrl + /

5. Paragraph: The "< p >" (paragraph) tag in HTML is used to define and display a paragraph of text on a web page. It automatically creates space before and after the paragraph, making the content easier to read.
